# Programming Test – Screening Round  
Author: Sagar Akolkar  
Language Used: Java  

This repository contains solutions for the 4 programming questions given in the screening test.  
Each problem is solved in a separate Java file.

## Problem 1 – Simple Calculator  
A basic calculator program that performs addition, subtraction, multiplication and division using a class and switch-case.  
File: Problem1.java

## Problem 2 – Odd Number Series  
Given an integer a, the program prints the first a odd numbers.  
Example:  
Input = 4 → Output = 1 3 5 7  
File: Problem2.java

## Problem 3 – Special Odd Pattern  
If the input number is odd, it prints odd numbers up to that number.  
If the input number is even, it prints odd numbers up to the previous odd number.  
File: Problem3.java

## Problem 4 – Count Multiples  
Given a list of numbers, the program counts how many numbers are divisible by 1 to 9.  
File: Problem4.java

## How to Run  
1. Open Eclipse or any Java IDE  
2. Create a new Java Project  
3. Add all the files inside the src folder  
4. Right-click any file → Run As → Java Application  

## Notes  
- Programs are written using basic Java  
- No external libraries used  
- Logic is kept simple for easy understanding
